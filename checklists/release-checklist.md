# Release Checklist

Source: `SYSTEM_PROMPT.md` §53-54, `claude/RELEASE.md`.

- ☐ Lint / format / static analysis passed
- ☐ Unit + integration tests passed
- ☐ Build succeeded
- ☐ Security scan passed
- ☐ Artifact generated
- ☐ Deployment validation passed
- ☐ Version number assigned
- ☐ Changelog written
- ☐ Migration notes written (if applicable)
- ☐ Known issues documented
- ☐ Rollback strategy confirmed viable
- ☐ Release is reproducible from a clean checkout

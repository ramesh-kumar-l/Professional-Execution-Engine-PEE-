## Summary

Design language: professional, minimal, elegant, information-dense, highly readable, keyboard-friendly, dark-mode-first with light mode equally polished. Target feel: the best qualities of Linear, Raycast, Notion, GitHub, VS Code, Figma, Cursor — without copying them (§92).

Component hierarchy: Tokens → Primitives → Components → Patterns → Templates → Screens → Workflows (§94).

# Accessibility Checklist

Source: `SYSTEM_PROMPT.md` §97.

- ☐ Full keyboard navigation
- ☐ Screen-reader support
- ☐ High-contrast mode supported
- ☐ Reduced-motion respected
- ☐ Visible focus indicators
- ☐ Semantic HTML used
- ☐ Appropriate ARIA usage (no ARIA where native semantics suffice)
- ☐ Color is never the only signal (errors/warnings also use text/icon)
- ☐ Validated with keyboard-only and screen-reader passes, not just visually

# session/ — Live Session State

Lightweight, moment-to-moment session tracking — updated *during* a session, unlike `project-memory-bank/28-session-handoff.md` which captures only the final handoff.

**When to load:** `current-session.md` at the start of any session, to see if one is already in progress.

**When to update:** continuously during a session; finalized into `project-memory-bank/28-session-handoff.md` and `29-next-task.md` at session end (`SYSTEM_PROMPT.md` §31, §87).

## Contents

`current-session.md`, `today-plan.md`, `completed.md`, `next-task.md`, `session-handoff.md`.

Populated in Group 10, updated continuously during each session thereafter.
